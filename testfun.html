HTML Integration

Add furniture booking to any website with simple HTML attributes:

<!-- Load the widget --> <script src="https://your-cdn.com/universal-booking-widget.min.js"></script> <!-- Basic furniture booking button --> <button data-universal-booking="furniture"> Book Furniture </button> <!-- With custom configuration --> <button data-universal-booking="furniture" data-config='{"locationId": 1, "branding": {"primaryColor": "#f97316"}}'> Reserve Cabana </button>

JavaScript Integration

Programmatically control the furniture booking widget:

// Initialize furniture booking widget const furnitureWidget = UniversalBookingWidget.init({ businessType: 'furniture', locationId: 1, apiBaseUrl: 'https://your-api.com/api', branding: { primaryColor: '#f97316', companyName: 'Nike Lake Resort' } }); // Open the widget furnitureWidget.open(); // Control widget lifecycle furnitureWidget.close(); furnitureWidget.destroy();

React Integration

Use with React applications:

import { UniversalBookingWidget } from 'universal-booking-widget'; function FurnitureBookingButton() { const openFurnitureBooking = () => { UniversalBookingWidget.init({ businessType: 'furniture', locationId: 1, apiBaseUrl: process.env.REACT_APP_API_URL, branding: { primaryColor: '#f97316' } }).open(); }; return ( <button onClick={openFurnitureBooking}> Book Furniture </button> ); }

Your backend must implement these endpoints for furniture booking:

GET /booking/furniture/furniture-list?location_id=1
Returns list of available furniture types
GET /booking/furniture/sessions?date=2025-09-27&furniture_id=1&platform=web
Returns available time slots for selected furniture and date
POST /booking/furniture/availability
Check availability for specific furniture, date, and session
Body: {"furniture_id": 1, "date": "2025-10-01", "session_id": 1}
POST /booking/furniture/bookings?location_id=1
Create new furniture booking with customer info and payment
